Gamma Intelligence

A domain-specific AI assistant built using:

- RAG architecture
- FAISS vector database
- SentenceTransformers embeddings
- Llama 3 via Groq API
- Streamlit frontend

## Features

- ChatGPT-style interface
- Domain-restricted answers
- Context-aware retrieval
- Temperature control
- Chat memory
